# Space Travelers Hub

Web application that uses React and Redux to keep track of a list of missions ands rockets that the user can reserve and show them on their profile page.

## Built With

- Javascript
- React
- Redux
- HTML
- CSS

### Prerequisites

- Text Editor (VSCode for example)
- Node and npm installed
- Browser

## Getting Started - How to deploy on your machine

- Clone the project
- run `npm install` at your command line to install al dependencies
- run `npm start`
